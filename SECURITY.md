# Security Policy

## Reporting a vulnerability

Please report security vulnerabilities privately.  Do **not** open a public
issue for security problems.

- Preferred: use the forge's private vulnerability reporting if available.
- Email: vuln@mvct.com

Include enough detail to reproduce the issue (steps, environment, and impact).
We will acknowledge your report, investigate, and keep you informed of progress
toward a fix.

## Supported versions

The latest released version on the `main` branch receives security fixes.
